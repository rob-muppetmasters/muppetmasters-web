# MuppetMasters Web

## Tagline
**RETRO. REWIND. RE-PLAY.**

A retro-themed website celebrating classic computing and vintage games.

## Overview

MuppetMasters is a nostalgic web project dedicated to retro computing culture. The site features a distinctive retro aesthetic with neon styling, scanline effects, and a terminal-inspired design reminiscent of classic computer systems.

## Features

### Design & Visual Style
- **Retro Aesthetic**: Neon blue and pink color scheme with black background
- **Scanline Effects**: Vintage monitor-style visual effects overlay
- **Terminal Font**: Courier New monospace font for authentic retro feel
- **Responsive Layout**: Sidebar navigation with main content area

### Color Palette
- Neon Blue: `#00f2ff`
- Neon Pink: `#ff00ff`
- Terminal Green: `#33ff33`
- Background Black: `#0b0b0e`
